# 🧾 Billing & Invoice System

A full-featured **Billing and Invoice Management System** built using **ASP.NET Core MVC** with **Entity Framework Core**.

This system allows Admins and Staff to manage customers, products, and generate professional invoices with discount support and PDF export.

---

## 🚀 Live Demo: ### http://billingandinvoice.somee.com/User/Login

---

## 🚀 Features

### 👨‍💼 Admin Panel

* Dashboard with:

  * Total invoices
  * Total revenue
  * Filters (date, staff, search)
* Manage Customers (Add / Edit / Delete)
* Manage Products
* Manage Staff Users
* View all invoices
* Filter invoices by:

  * Customer name
  * Invoice number
  * Staff/Admin (Biller)
  * Date range

---

### 👨‍🔧 Staff Panel

* Personal dashboard (only their invoices)
* Create billing
* Select customers
* Manage cart
* Generate invoices

---

### 🧾 Invoice System

* Auto invoice generation
* Unique invoice number
* Biller name (Admin/Staff)
* Discount support:

  * Fixed amount
  * Percentage
* Final amount calculation
* Printable invoice
* PDF download (Rotativa)

---

### 🛒 Cart System

* Add products to cart
* Update quantity
* Remove items
* Auto total calculation

---

### 👤 Customer Management

* Add customer
* Edit customer
* Delete customer (Admin only)
* Search customer
* Select customer for billing

---

### 🔐 Authentication

* Login system (Admin / Staff)
* Session-based authentication
* Forgot password feature
* Role-based access control

---

### 🎨 UI Features

* Dark themed navbar & footer
* Responsive design (Bootstrap)
* Dashboard UI
* Attractive invoice layout
* Modern buttons & gradients

---

## 🛠️ Technologies Used

* ASP.NET Core MVC (.NET 6)
* Entity Framework Core
* SQL Server
* Bootstrap 5
* jQuery
* Rotativa.AspNetCore (PDF generation)

---

## 🔑 Default Roles

* Staff → Limited access
